project.conf
============

Project configuration file (each project has one).

``[project]`` Section Settings
------------------------------

Project global configuration.

``[project]`` Section Values
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

``name``

  Project verbose name.

  *Default*: The project identifier.

  *Required*: No.

  *Introduced*: 0.1.0

-------------------------------
