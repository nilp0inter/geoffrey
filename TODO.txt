Features & Tasks
================

* Plugin loader.
  * Plugin loader can found plugins.
  * The project should initialize the configured plugins.
* Filesystem plugin.
  * Filesystem plugin can detect file changes.
  * Filesystem plugin can monitorize multiple paths.
* Event HUB.
  * Plugins can send events to the event hub.
  * Plugins can send states to the event hub.
  * The hub should convert states to events when neccesary.
  * Plugins can subscribe to events determined by filters.
  * Events should persist restarts.
  * States should persist restarts.
